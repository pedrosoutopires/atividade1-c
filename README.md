# Atividades de Programação em C

Este repositório contém exercícios desenvolvidos durante a disciplina de programação em C.

## Lista de Exercícios

## Escrever códigos de programas

1. Cálculo da área de um quadrado  
2. Cálculo de reajuste salarial  
3. Cálculo de ferraduras para cavalos  
4. Arrecadação de padaria (pães e broas)  
5. Decomposição de valor em moedas  
6. Conversão de idade (anos, meses e dias → dias)  
7. Conversão de idade (dias → anos, meses e dias)  
8. Cálculo de litros de gasolina  
9. Custo final de um carro  
10. Média ponderada de notas  
11. Venda de camisetas  
12. Cálculo de salário com aumento e impostos  
13. Divisão de conta entre amigos  
14. Cálculo de salário com horas extras  
15. Cálculo de desconto em produto  
16. Mistura de refresco (água e suco)  
17. Cálculo da altura de um prédio por sombras  

## Funções

1. Converter celsius para fahrenheit
2. Calcular a área de um circulo
3. Converter horas e minutos para minutos
4. Calcular hipotenusa

## Objetivo

Praticar conceitos básicos da linguagem C, como:
- Entrada e saída de dados
- Operações matemáticas
- Uso de variáveis
